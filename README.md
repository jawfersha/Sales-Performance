🍫 Chocolate Sales Data Analytics Project
📌 Overview

This project is an end-to-end data analytics solution built using a Kaggle dataset containing 5 CSV files. The goal was to analyze sales performance, uncover business insights, and build an interactive dashboard for decision-making.

The workflow covers data preparation, SQL analysis, and visualization using Power BI.

📂 Dataset
Source: Kaggle – Chocolate Store Sales
Tables: 5 (Sales, Products, Customers, Stores, Calendar)
Records: 10,500 transactions
Scope: 6 countries, 4 store types, 202 products
🛠️ Tools & Technologies
SQL (PostgreSQL) – Data cleaning, joins, analysis
Power BI – Dashboard & visualization
Python (Pandas) – Data preprocessing
🏗️ Data Modeling
Designed a star schema
Fact Table: sales
Dimension Tables: products, customers, stores, calendar
Performed multi-table joins to create a unified dataset
📊 Key Analysis Performed
Revenue, Cost, and Profit calculation
Monthly and yearly sales trends
Top products and top customers
Store and country performance
Profit margin analysis (~39.9%)
Customer segmentation (gender & loyalty)
Weekday vs weekend sales
📈 Power BI Dashboard

The dashboard includes:

KPI Cards: Revenue, Profit, Orders, Units Sold, Avg Order Value
Monthly revenue & profit trends
Category-wise sales distribution
Top-performing products
Sales by country and store type
Customer segmentation
🔍 Key Insights
Total Revenue: $268K+ with ~40% profit margin
Top Product: Dark Chocolate 50% (highest revenue & profit)
Best Month: May 2023
Top Market: Canada
Airport stores generate highest revenue
💡 Business Recommendations
Focus marketing on high-performing products
Expand high-revenue store types (e.g., airport stores)
Improve pricing for low-margin products
Strengthen customer loyalty programs
🚀 Skills Demonstrated
SQL (Joins, Aggregations, Business Queries)
Data Modeling (Star Schema)
Data Analysis & Business Insights
Power BI (DAX, Dashboard Design, Visualization)
📎 Conclusion

This project demonstrates the complete data analytics workflow—from raw data to actionable insights—using industry-relevant tools and techniques.
